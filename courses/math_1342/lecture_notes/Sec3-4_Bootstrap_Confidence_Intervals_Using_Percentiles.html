Section 3.4

Bootstrap Confidence Intervals
Using Percentiles

A flexible method for any confidence level — just keep the middle P%

Motivation

Beyond 95% Confidence

What if we want a 90% CI or a 99% CI? The SE method used "2" specifically for 95%.

Recall: SE Method (Section 3.3)

\[\text{95% CI} = \text{Statistic} \pm 2 \times \text{SE}\]

The multiplier "2" works specifically for 95% confidence with a bell-shaped distribution.

What multiplier do we use for 90%? 99%? Other levels?

A More General Approach

  • Instead of a formula with a multiplier, we can read the CI directly from the bootstrap distribution
  • This is the percentile method
  • It works for any confidence level
  • No need to remember different multipliers
Method

The Percentile Method

For a P% confidence interval, keep the middle P% of bootstrap statistics.

The Rule

For a P% CI: cut off \(\frac{100 - P}{2}\)% from each tail of the bootstrap distribution

Common Confidence Levels

Confidence Level Cut from Each Tail Lower Percentile Upper Percentile
90% 5% 5th percentile 95th percentile
95% 2.5% 2.5th percentile 97.5th percentile
99% 0.5% 0.5th percentile 99.5th percentile
Example: For a 95% CI, find the 2.5th and 97.5th percentiles of your bootstrap distribution. Those are your lower and upper bounds.
Relationship

Confidence Level and Interval Width

How does changing the confidence level affect the interval?

90% CI

Narrow

Keep middle 90%. Cut 5% from each tail. Less confident, but more precise.

95% CI

Medium

Keep middle 95%. Cut 2.5% from each tail. The standard choice.

99% CI

Wide

Keep middle 99%. Cut 0.5% from each tail. Very confident, but less precise.

The trade-off: Higher confidence level \(\Rightarrow\) wider interval. To be more confident we capture the parameter, we need to cast a wider net.
Sample Size

Sample Size and Interval Width

How does the sample size affect the width of the confidence interval?

Smaller Sample Size

  • More variability in the statistic
  • Larger SE
  • Bootstrap distribution is more spread out
  • Results in a wider confidence interval

Larger Sample Size

  • Less variability in the statistic
  • Smaller SE
  • Bootstrap distribution is more concentrated
  • Results in a narrower confidence interval
Summary of what affects width:
Higher confidence level \(\Rightarrow\) wider interval
Larger sample size \(\Rightarrow\) narrower interval
Comparison

Two Methods for Bootstrap CIs

SE method vs. percentile method — when to use which?

SE Method

\[\text{Statistic} \pm 2 \times \text{SE}_{\text{bootstrap}}\]
  • Uses a multiplier (2 for 95%)
  • Assumes the bootstrap distribution is bell-shaped and symmetric
  • Works well for 95% confidence
  • Less flexible for other confidence levels

Percentile Method

Keep the middle P% of bootstrap statistics
  • Read bounds directly from the bootstrap distribution
  • Works for any confidence level
  • Does not require a symmetric distribution
  • More flexible and widely applicable
When they agree: For a symmetric, bell-shaped bootstrap distribution at the 95% level, both methods give very similar results. For non-bell-shaped distributions or non-95% levels, use the percentile method.
Cautions

Bootstrap Cautions

The bootstrap is powerful, but it has limitations.

When to Be Careful

  • The bootstrap distribution should look smooth and roughly symmetric (bell-shaped)
  • If the bootstrap distribution is very skewed or has gaps, the CI may not be reliable
  • Very small samples may not produce trustworthy bootstrap distributions

Other Considerations

  • The original sample must be representative of the population
  • Bootstrapping cannot fix a biased sample
  • Use enough bootstrap samples (at least 1000, ideally more)
  • Always plot the bootstrap distribution to check its shape
Bottom line: The bootstrap works well when the bootstrap distribution is smooth and symmetric. Always visualize it before trusting the results.

  • 95% CI: 2.5th to 97.5th percentile
  • 99% CI: 0.5th to 99.5th percentile

  • Use percentile method for non-95% levels or non-symmetric distributions

  • Always plot the bootstrap distribution to check
  • The bootstrap cannot fix a biased or unrepresentative sample
