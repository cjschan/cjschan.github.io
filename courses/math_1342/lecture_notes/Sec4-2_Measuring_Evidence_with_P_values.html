Measuring Evidence
with P-values

How do we quantify the strength of evidence against the null hypothesis?

Measuring Evidence Against \(H_0\)

What counts as convincing evidence?

The Core Question

  • We have a sample statistic from our data
  • To decide if this statistic provides evidence against \(H_0\), we need to know: what would we observe just by random chance if \(H_0\) were true?
  • If our observed statistic would be very unusual under \(H_0\), that is evidence against \(H_0\)

The Key Idea

  • Simulate many samples assuming \(H_0\) is true
  • Compute the statistic for each simulated sample
  • See where our observed statistic falls relative to the simulated values
  • If our statistic is in the tail, that is evidence against \(H_0\)

Randomization Distribution

Simulating a world where \(H_0\) is true

Definition

  • A randomization distribution is a collection of statistics from samples simulated assuming \(H_0\) is true
  • It shows us what values of the statistic would occur by random chance alone
  • This distribution is centered at the null hypothesis value

How to Build One

  • Step 1: Assume \(H_0\) is true
  • Step 2: Simulate a random sample under that assumption
  • Step 3: Compute the sample statistic
  • Step 4: Repeat steps 2-3 many times (e.g., 1000+)
  • Step 5: Plot all the simulated statistics
The randomization distribution provides a benchmark: it tells us what "random chance" looks like so we can judge whether our observed result is surprising.

Paul the Octopus

A famous example of measuring evidence

The Story

  • Paul the Octopus correctly predicted all 8 knockout-round games in the 2010 World Cup
  • Was Paul psychic, or just lucky?
  • \(H_0: p = 0.5\) (Paul is just guessing)
  • \(H_a: p > 0.5\) (Paul can predict)

Simulating Under \(H_0\)

  • If Paul is guessing (\(p = 0.5\)), each prediction is like a fair coin flip
  • Simulate: flip a coin 8 times, count heads
  • Repeat 1000 times and build a distribution
  • Getting 8 out of 8 correct is extremely rare under random guessing
Only about 4 out of 1000 simulations gave 8/8 correct  →  p-value \(\approx 0.004\)
A p-value of 0.004 means there is only a 0.4% chance of getting a result this extreme if \(H_0\) is true. This is strong evidence against \(H_0\).

The p-value

The formal measure of evidence against \(H_0\)

Definition

The p-value is the probability of obtaining a sample statistic as extreme (or more extreme) than the one observed, assuming \(H_0\) is true.

Key Properties

  • The p-value is always between 0 and 1
  • A small p-value means the observed result is unlikely under \(H_0\) → evidence against \(H_0\)
  • A large p-value means the observed result is plausible under \(H_0\) → no evidence against \(H_0\)

Common Misconception

  • The p-value is NOT the probability that \(H_0\) is true
  • It is the probability of seeing data this extreme if \(H_0\) were true
  • Think of it as: "How surprising is our data under \(H_0\)?"

Computing p-values

Using the randomization distribution to find the p-value

From a Randomization Distribution

  • The p-value is the proportion of statistics in the randomization distribution that are as extreme as (or more extreme than) the observed statistic
  • The direction of "extreme" depends on \(H_a\)
\(\text{p-value} = \frac{\text{# simulated statistics as extreme as observed}}{\text{total # of simulations}}\)

Direction Matters

  • If \(H_a\) uses \(>\): count simulated values ≥ observed (upper tail)
  • If \(H_a\) uses \(<\): count simulated values ≤ observed (lower tail)
  • If \(H_a\) uses \(\neq\): count values in both tails beyond the observed

Example: Tea and Immunity

Can drinking tea boost the immune system compared to coffee?

The Study

  • Subjects randomly assigned to drink tea or coffee for several weeks
  • Immune response measured via interferon gamma production
  • \(H_0: \mu_t = \mu_c\) (no difference in mean immune response)
  • \(H_a: \mu_t > \mu_c\) (tea drinkers have higher immune response)

Results

  • Observed difference in means: \(\bar{x}_t - \bar{x}_c = 3.5\)
  • Randomization distribution built by reassigning groups randomly 1000 times
  • Only 26 out of 1000 simulations produced a difference ≥ 3.5
p-value \(= \frac{26}{1000} = 0.026\)
A p-value of 0.026 provides fairly strong evidence that tea boosts immune response more than coffee. Only 2.6% of random reassignments produced a difference this large.

One-Sided vs. Two-Sided p-values

How the direction of \(H_a\) affects the p-value calculation

Upper-Tail Test

\(H_a: \text{parameter} > \text{value}\)

  • p-value = proportion in right tail beyond the observed statistic
  • Count simulated values ≥ observed

Lower-Tail Test

\(H_a: \text{parameter} < \text{value}\)

  • p-value = proportion in left tail beyond the observed statistic
  • Count simulated values ≤ observed

Two-Tailed Test

\(H_a: \text{parameter} \neq \text{value}\)

  • p-value = proportion in both tails beyond the observed statistic
  • Count extreme values on both sides
For a two-sided test:   p-value \(= 2 \times\) (proportion in the smallest tail)
Example (Green Tea & Prostate Cancer): Researchers found a significant reduction in cancer risk among green tea drinkers. With a one-sided test, the p-value was 0.0005 -- extremely strong evidence.

Summary: Measuring Evidence with P-values

Key takeaways from Section 4.2

Core Concepts

  • A randomization distribution shows what statistics look like when \(H_0\) is true
  • The p-value measures the probability of results as extreme as observed, if \(H_0\) is true
  • Small p-values = strong evidence against \(H_0\)
  • Large p-values = data are consistent with \(H_0\)

Computing p-values

  • p-value = proportion of randomization statistics as extreme as observed
  • Direction depends on \(H_a\): upper, lower, or two-tailed
  • For two-sided: p-value = \(2 \times\) smallest tail proportion
  • The p-value is not the probability that \(H_0\) is true
