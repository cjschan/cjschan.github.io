Section 3.3

Constructing Bootstrap
Confidence Intervals

Using resampling to estimate the standard error when we only have one sample

The Problem

We Only Have One Sample

To build a confidence interval, we need the SE. But how do we get it?

What We Need

  • A 95% CI requires: Statistic \(\pm\; 2 \times \text{SE}\)
  • SE = standard deviation of the sampling distribution
  • The sampling distribution comes from taking many samples from the population

What We Have

  • In practice, we only have one sample
  • We cannot go back to the population to take more samples
  • So we cannot directly compute the sampling distribution or the SE
The question: How can we estimate the variability of a statistic if we only have a single sample? The answer: the bootstrap.
Key Idea

The Bootstrap Idea

Resample from your one sample — with replacement — to mimic drawing new samples.

Core insight: pretend the sample IS the population. If the sample is representative, resampling from it with replacement mimics drawing fresh samples — no new data collection required.
Definitions

Bootstrap Samples and Statistics

Precise definitions of the key bootstrap terms.

Bootstrap Sample

  • A random sample drawn with replacement from the original sample
  • Must be the same size (\(n\)) as the original sample
  • Some original values will appear more than once; others may not appear at all

Bootstrap Statistic

  • The statistic (mean, proportion, median, etc.) computed from a bootstrap sample
  • Each bootstrap sample gives one bootstrap statistic
  • Example: \(\bar{x}^*\) = mean of a bootstrap sample
"With replacement" is essential. It means after drawing an observation, we put it back so it can be drawn again. This is what creates variability between bootstrap samples.
Distribution

The Bootstrap Distribution

Collecting many bootstrap statistics to see how much they vary.

Definition

  • The bootstrap distribution is the distribution of many bootstrap statistics, each computed from a different bootstrap sample

How to Build It

  • Take a bootstrap sample (size \(n\), with replacement)
  • Compute the bootstrap statistic
  • Repeat this process many times (e.g., 1000+ times)
  • Plot the distribution of all bootstrap statistics

What It Tells Us

  • Shows the shape of the sampling distribution
  • Shows the spread (variability) of the statistic
  • Allows us to estimate the SE
  • All from just a single original sample
Golden Rule

The Golden Rule of Bootstrapping

The key analogy that makes the bootstrap work.

The Analogy

Bootstrap statistics are to the sample statistic
as
sample statistics are to the population parameter

Real World

  • Population \(\xrightarrow{\text{sample}}\) Sample statistic
  • The sampling distribution is centered at the population parameter
  • The spread of the sampling distribution = SE

Bootstrap World

  • Original sample \(\xrightarrow{\text{resample}}\) Bootstrap statistic
  • The bootstrap distribution is centered at the sample statistic
  • The spread of the bootstrap distribution estimates SE
Important: The bootstrap distribution is centered around the sample statistic, not the population parameter. But its spread estimates the true SE.
Estimation

Estimating the Standard Error

The SD of the bootstrap distribution gives us the SE we need.

The Key Result

\[\text{SE} \approx \text{SD of bootstrap distribution}\]

Compute the standard deviation of all your bootstrap statistics — that is your estimated standard error.

Why This Works

  • The bootstrap distribution mimics the shape and spread of the true sampling distribution
  • The variability among bootstrap statistics estimates the variability among sample statistics
  • More bootstrap samples = better estimate of SE

Works for ANY Statistic

  • Mean, proportion, median, standard deviation, correlation, difference in means...
  • The bootstrap doesn't need a formula for the SE
  • Just resample, compute, and look at the spread
Computation

Bootstrap 95% Confidence Interval (SE Method)

Plugging the bootstrap SE into our familiar formula.

The Formula

\[\text{95% CI} = \text{Statistic} \pm 2 \times \text{SE}_{\text{bootstrap}}\]

Step-by-Step

  • 1. Start with your original sample and compute the statistic
  • 2. Generate many bootstrap samples (with replacement, same size \(n\))
  • 3. Compute the bootstrap statistic for each
  • 4. Find the SD of all bootstrap statistics = SE
  • 5. 95% CI = Statistic \(\pm\; 2 \times\) SE

Example

Original sample mean: \(\bar{x} = 25.3\)

SD of 1000 bootstrap means: 1.8

\(\text{SE} \approx 1.8\)

\(95\%\;\text{CI} = 25.3 \pm 2(1.8)\)
\(= (21.7,\; 28.9)\)

Recap

Section 3.3 Summary

The Bootstrap Process

  • We only have one sample, so we resample with replacement from it
  • Each resample is the same size \(n\) as the original
  • Compute a statistic on each resample = bootstrap statistic
  • The collection of bootstrap statistics = bootstrap distribution

Key Results

  • Bootstrap distribution is centered at the sample statistic
  • SD of bootstrap distribution estimates the SE
  • 95% CI = Statistic \(\pm\; 2 \times\) SE
  • Works for any statistic — no formula needed
Golden Rule: Bootstrap statistics are to the sample statistic as sample statistics are to the population parameter. The bootstrap distribution's spread estimates how much the statistic varies.
