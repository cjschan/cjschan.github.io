Section 4.5

Making Connections

Connecting randomization distributions, bootstrap distributions, confidence intervals, and hypothesis tests.

Generating Randomization Samples

Three criteria for creating samples under the null hypothesis.

1
Be consistent with the null hypothesis.
2
Use the data in the original sample.
3
Reflect the way the original data were collected.

Methods Vary by Situation

The specific simulation method depends on the study design.

Randomized Experiment

  • Re-randomize cases to treatment groups, keeping response values fixed.
  • Example: cocaine addiction study.

Observational Study

  • Shift data to make \(H_0\) true, then bootstrap.
  • Example: body temperature (shift all values so the mean equals 98.6).
For a single proportion, simulate coin flips. The general idea is always the same: simulate what would happen by random chance if \(H_0\) were true.

Bootstrap vs. Randomization Distributions

They look similar but serve different purposes.

Bootstrap Distribution

  • Does not assume \(H_0\) is true.
  • Centered around the sample statistic.
  • Used for confidence intervals.

Randomization Distribution

  • Assumes \(H_0\) is true.
  • Centered around the null value.
  • Used for hypothesis tests.
The two distributions have the same shape and spread. The only difference is the center.

Confidence Intervals and Hypothesis Tests

A confidence interval and a two-sided test give consistent conclusions.

The Connection

  • If the null value is outside a 95% CI, a two-tailed test at \(\alpha = 0.05\) will reject \(H_0\).
  • If the null value is inside a 95% CI, a two-tailed test at \(\alpha = 0.05\) will not reject \(H_0\).

When to Use Which

  • Confidence intervals: when you want to estimate a parameter.
  • Hypothesis tests: when you want to test a claim about a parameter.

What to Remember

Everything connects.

Randomization Samples

  • Consistent with \(H_0\)
  • Use observed data
  • Reflect data collection method

CI and Test Agreement

  • Null value outside 95% CI \(\Rightarrow\) reject \(H_0\) at \(\alpha = 0.05\)
  • Null value inside 95% CI \(\Rightarrow\) do not reject \(H_0\)
