The Structure of Data

Data are everywhere. Understanding how data are organized is the first step in statistics.

What Is Data?

Information collected from observations, surveys, and experiments.

Collect
Gather information through surveys, experiments, or observation.
Describe
Summarize and visualize the information you gathered.
Analyze
Draw conclusions and make decisions based on the data.

Cases and Variables

A dataset is usually organized so that each row is a case and each column is a variable.

Cases

  • The individual subjects or objects we collect information about.
  • Each row in a dataset represents one case.
  • Examples: a person, a country, a movie.

Variables

  • A characteristic recorded for each case.
  • Each column in a dataset represents one variable.
  • Examples: age, gender, income, test score.

Categorical vs. Quantitative Variables

The type of variable determines which summaries and visualizations we use.

Categorical

  • Divides cases into groups or categories.
  • Each case falls into exactly one category.
  • Examples: gender, eye color, political party, yes/no responses.

Quantitative

  • Records a numerical quantity for each case.
  • Arithmetic operations make sense on these values.
  • Examples: height, weight, temperature, test score.
Caution: Think carefully about what the cases are. The same topic can yield a categorical or quantitative variable depending on how it is recorded.

Explanatory and Response Variables

When we use one variable to predict or explain another, each gets a name.

Explanatory Variable

The variable we think might influence or predict the other.
  • Does meditation help reduce stress? Explanatory: meditation
  • Does sugar cause hyperactivity? Explanatory: sugar consumption

Response Variable

The variable we are trying to understand or predict.
  • Does meditation help reduce stress? Response: stress level
  • Does sugar cause hyperactivity? Response: hyperactive behavior

What to Remember

The building blocks of every statistical study.

Data Structure

  • Data are collected on cases (rows).
  • Characteristics are recorded as variables (columns).
  • A dataset organizes cases and variables into a table.

Variable Types

  • Categorical variables place cases into groups.
  • Quantitative variables measure numerical quantities.
  • An explanatory variable predicts a response variable.
