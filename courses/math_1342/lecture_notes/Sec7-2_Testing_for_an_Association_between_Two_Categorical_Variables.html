Section 7.2

Testing for an Association Between Two Categorical Variables

Use a chi-square test of association to decide whether two categorical variables are related in the population.

What This Test Does

We compare the observed two-way table to what we would expect if the variables were not associated.

When to Use It

  • Both variables are categorical.
  • Data are summarized in a two-way table.
  • Goal is to test for association, not causation.

Null and Alternative

\[ H_0:\text{the variables are not associated} \]
\[ H_a:\text{the variables are associated} \]

Observed Counts Example

The StudentSurvey data compare award preference and which SAT score is higher.

Observed Table

Higher SATAcademyNobelOlympicTotal
Math2168116205
Verbal104595150
Total31113211355
Question
Does preferred award appear related to whether a student has the higher Math or Verbal SAT score?
Idea Under \(H_0\)
If there is no association, then each award-preference column should have about the same Math/Verbal proportions as the overall sample.

Expected Counts

Expected counts come from row total times column total divided by the grand total.

General Formula

\[ Expected = \frac{(\text{row total})(\text{column total})}{\text{grand total}} \]
This uses the assumption that the row and column variables are unrelated under the null hypothesis.

Example Cell

Expected count for Academy and Math:
\[ \frac{205 \cdot 31}{355} \approx 17.90 \]
The observed count is \(21\), so this cell contributes some evidence against \(H_0\).

Expected Table for the Example

Compute an expected count for every cell before finding the chi-square statistic.

Condition

  • All expected counts should be at least 5.
  • Counts, not percentages, must go in the table.
  • Randomization or independent sampling still matters.

Chi-Square Statistic

Each cell contributes a nonnegative amount. Large total contributions suggest association.

Cell Contribution

\[ \frac{(O-E)^2}{E} \]
Example for the Academy/Math cell:
\[ \frac{(21-17.90)^2}{17.90}\approx 0.54 \]

Total Statistic

\[ \chi^2 \approx 2.10 \]
This is found by adding all six cell contributions in the table.

Degrees of Freedom and p-Value

For a two-way table, the degrees of freedom depend on the number of rows and columns.

Degrees of Freedom

\[ df=(r-1)(c-1) \]
For a \(2\times 3\) table:
\[ df=(2-1)(3-1)=2 \]

p-Value

A chi-square statistic of about \(2.10\) with \(df=2\) gives a p-value much larger than \(0.05\), so this example does not show strong evidence of association.
The p-value is always a right-tail area because larger chi-square values mean a bigger departure from \(H_0\).

This table shows a much larger departure from independence than the SAT example, so it produces a much smaller p-value.

Special Case: \(2\times 2\) Tables

When each variable has two categories, a chi-square test and a two-proportion z test are closely related ways to analyze the same table.

Same Question

  • Both methods test whether the two categorical variables are related.
  • Both use the same observed \(2\times 2\) table.
  • For the same data, they lead to the same two-sided conclusion.

Interpretation

  • The z test is often introduced first for two proportions.
  • The chi-square method is more general because it extends beyond two categories.
  • For larger tables, use chi-square rather than a z test.

What You Can and Cannot Conclude

A significant result shows association, but it does not automatically show causation.

Association
The variables appear related in the population.
Not Direction
The test does not tell you which variable explains the other.
Not Causation
Without random assignment, a relationship may be due to confounding.

The Chi-Square Distribution

For \(df = 4\). The right tail beyond an observed \(\chi^2 = 9.49\) is the p-value.

What to Remember

A chi-square test of association compares observed counts to expected counts from independence.

Core Steps

  • State \(H_0\) and \(H_a\).
  • Find each expected count.
  • Compute \(\chi^2=\sum (O-E)^2/E\).
  • Use \(df=(r-1)(c-1)\).

Conditions

  • Two categorical variables.
  • Independent observations or randomization.
  • All expected counts at least 5.
  • Conclusion is about association, not causation.
