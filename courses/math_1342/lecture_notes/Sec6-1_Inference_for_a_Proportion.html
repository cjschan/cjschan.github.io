Section 6.1

Inference for a Proportion

Confidence intervals first, then one-proportion z tests.

Confidence Intervals First

Start with the same logic from the randomization approach, then switch to the normal model.

  • Earlier, we used a randomization plot to build a confidence interval.
  • We could read the standard error from the spread of that plot.
  • For a 95% confidence interval, we were essentially using \(z^* \approx 2\).
  • For other confidence levels, we used percentiles from the randomization distribution.
95% CI
Same idea as before. The shortcut now is the normal distribution.

Normal Model for Confidence Intervals

With large samples, we use the standard normal distribution instead of simulated cutoffs.

General Form

\[ \text{Confidence interval} = \text{sample statistic} \pm z^* \cdot SE \]
\(z^*\) changes with the confidence level.
  • \(90\%\): about \(1.645\)
  • \(95\%\): about \(1.96\)
  • \(99\%\): about \(2.576\)

What Changes?

  • \(z^*\) comes from the normal distribution.
  • The standard error formula depends on the statistic.
  • For a proportion, the SE formula differs from the one for a mean.

Confidence Interval for One Proportion

For a confidence interval, use the sample proportion in the SE formula.

Formula

\[ SE(\hat{p})=\sqrt{\frac{\hat{p}(1-\hat{p})}{n}} \]
For a confidence interval, use the sample value \(\hat{p}\) because it is our best estimate of the true population proportion.

Conditions

  • Need a large enough sample.
  • Check that counts in both categories are at least 10.
  • Then the sampling distribution of \(\hat{p}\) is approximately normal.

Confidence Interval Example

In a sample of \(200\) students, \(124\) say they prefer online notes.

Step 1
Compute the sample proportion. This gives the sample estimate of the population proportion.
\[ \hat{p}=\frac{124}{200}=0.62 \]
Step 2
Compute the standard error. This measures typical sample-to-sample variation in \(\hat{p}\).
\[ SE=\sqrt{\frac{0.62(1-0.62)}{200}}\approx 0.034 \]
Step 3
Use \(z^*=1.96\) for a \(95\%\) confidence interval and compute the margin of error.
\[ 0.62 \pm 1.96(0.034)=0.62 \pm 0.067 \]
Step 4
Add and subtract the margin of error from the sample proportion.
\[ (0.553,\ 0.687) \]
Interpretation: We are \(95\%\) confident that the true proportion of students who prefer online notes is between \(55.3\%\) and \(68.7\%\).

Hypothesis Testing Uses a z Score

A z test compares the sample result to the null value in standard error units.

General z Statistic

\[ z=\frac{\text{sample statistic}-\text{null value}}{SE} \]
The z score tells us whether the sample result would be unusual if the null hypothesis were true.

One-Proportion Test

\[ z=\frac{\hat{p}-p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}} \]
  • State \(H_0\) and \(H_a\).
  • Use the null value \(p_0\) in the SE formula.
  • Find the p-value from the normal curve.

Hypothesis Test Example

In a random sample of \(500\) US citizens, \(280\) plan to vote.

Step 1
Set up the hypotheses. We want evidence that more than half plan to vote, so this is right-tailed.
\[ H_0:p=0.50 \qquad H_a:p>0.50 \]
Step 2
Compute the sample proportion and the standard error using the null value \(p_0=0.50\).
\[ \hat{p}=\frac{280}{500}=0.56 \]
\[ SE=\sqrt{\frac{0.50(1-0.50)}{500}}\approx 0.0224 \]
Step 3
Compute the z statistic. This measures how many standard errors the sample is above the null value.
\[ z=\frac{0.56-0.50}{0.0224}\approx 2.68 \]
Step 4
Find the right-tail p-value and make the decision.
\[ \text{p-value}\approx 0.0036 \]
Conclusion: Reject \(H_0\). There is strong evidence that more than half of US citizens plan to vote.

How to Read the p-Value

The p-value is the tail area beyond the observed z score in the standard normal distribution.

  • Measure how far \(\hat{p}\) is from \(p_0\) in standard errors.
  • That distance is the z score.
  • The p-value is the tail area beyond that z score.
  • A small p-value gives evidence against \(H_0\).
\(z\)
Right-tailed, left-tailed, or two-tailed depends on the alternative hypothesis.

Sampling Distribution of a Sample Proportion

Theory says \(\hat{p}\) is approximately \(N(p, \sqrt{\frac{p(1-p)}{n}})\). For \(p = 0.4\) and \(n = 100\), \(SE \approx 0.049\).

What to Remember

Confidence intervals and hypothesis tests use similar ingredients, but not the same SE input.

Confidence Interval

\[ SE(\hat{p})=\sqrt{\frac{\hat{p}(1-\hat{p})}{n}} \]
  • Use \(\text{sample statistic} \pm z^* \cdot SE\).
  • For a proportion CI, use \(\hat{p}\) in the SE formula.
  • \(z^*\) comes from the confidence level.

Hypothesis Test

\[ SE(\hat{p})=\sqrt{\frac{p_0(1-p_0)}{n}} \]
  • Use a z score to compare the sample to the null value.
  • For a proportion test, use \(p_0\) in the SE formula.
  • The p-value is the tail area beyond the z score.
