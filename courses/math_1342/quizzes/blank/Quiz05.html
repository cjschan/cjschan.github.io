Quiz 5

Bootstrap CIs and Hypothesis Tests

Student Version

Covers Sections 3.3, 3.4, 4.1, 4.2

Instructions. This is a take-home quiz. You may use your lecture notes and online resources, but the work you submit must be your own. Show your work clearly to receive credit.

Total: 10 points

Problem 1 [2 pts]

Briefly describe how to construct one bootstrap sample from a sample of size \(n = 50\), and explain why bootstrapping is useful when we want a confidence interval but only have one sample. (1 pt each)

Problem 2 [2 pts]

A bootstrap distribution of the sample mean has standard error \(SE = 1.4\). The original sample mean is \(\bar{x} = 24.6\). Construct an approximate 95% confidence interval using the SE method (\(\bar{x} \pm 2 \, SE\)). (2 pts)

Problem 3 [2 pts]

1000 bootstrap means have 2.5th percentile \(= 21.6\) and 97.5th percentile \(= 27.5\). Give the percentile-method 95% CI. Briefly compare to the SE-method CI from Problem 2. (1 + 1)

Problem 4 [2 pts]

State an appropriate \(H_0\) and \(H_a\) for each scenario. (1 pt each)

  1. Researchers want evidence that more than 50% of voters in a city support a measure.
  2. A new drug should change average blood pressure (in either direction) compared with the current drug, where the current drug's average drop is 7 mmHg.

Problem 5 [2 pts]

A researcher runs a one-sided test and finds a p-value of 0.012.

  1. Interpret this p-value in context (assume \(H_0\) says \(p = 0.5\) and the alternative is \(p > 0.5\)). (1 pt)
  2. Using \(\alpha = 0.05\), what is the conclusion? Using \(\alpha = 0.01\)? (0.5 pt each)
