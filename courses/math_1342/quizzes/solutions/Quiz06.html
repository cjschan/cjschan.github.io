Quiz 6

Errors, Connections, and Normal-Based Tests

Worked Solutions

Covers Sections 4.4, 4.5, 5.1

Instructions. This is a take-home quiz. You may use your lecture notes and online resources, but the work you submit must be your own. Show your work clearly to receive credit.

Total: 10 points

Problem 1 [2 pts]

In a hypothesis test, define each in plain English. (1 pt each)

  1. Type I error
  2. Type II error

(a) Type I error: rejecting \(H_0\) when it is actually true (a false positive).

(b) Type II error: failing to reject \(H_0\) when it is actually false (a false negative).

Problem 2 [2 pts]

A 95% confidence interval for the mean is \((34.0,\ 39.0)\). Without doing a separate test, would you reject \(H_0: \mu = 36\) at \(\alpha = 0.05\)? Would you reject \(H_0: \mu = 40\)? Explain how the CI gives you this information.

Do not reject \(H_0: \mu = 36\) (36 lies inside the CI). Reject \(H_0: \mu = 40\) (40 lies outside the CI).

A two-sided test rejects \(H_0: \mu = c\) at level \(\alpha\) iff \(c\) is outside the corresponding \((1 - \alpha)\) CI. CIs and tests are two views of the same evidence.

Problem 3 [3 pts]

Compute the standardized test statistic. (1.5 pts each)

  1. For a single proportion: \(\hat{p} = 0.62\), \(p_0 = 0.55\), \(n = 200\). Use \(z = \frac{\hat{p} - p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}}\).
  2. For a single mean: \(\bar{x} = 102\), \(\mu_0 = 100\), \(s = 8\), \(n = 64\). Use \(z = \frac{\bar{x} - \mu_0}{s/\sqrt{n}}\).

(a) \(SE = \sqrt{\frac{0.55 \cdot 0.45}{200}} \approx 0.0352\). \(z = \frac{0.62 - 0.55}{0.0352} \approx 1.99\).

(b) \(SE = \frac{8}{\sqrt{64}} = 1\). \(z = \frac{102 - 100}{1} = 2\).

Problem 4 [3 pts]

The shaded region of the normal curve below shows the rejection region for a two-sided test at \(\alpha = 0.05\). (1 pt each)

  1. What z-values mark the boundaries?
  2. If the observed z is 1.5, what is the conclusion?
  3. If the observed z is 2.4, what is the conclusion?

(a) \(z = \pm 1.96\). (b) 1.5 is inside \((-1.96, 1.96)\), so do not reject \(H_0\). (c) 2.4 is outside that interval, so reject \(H_0\).
