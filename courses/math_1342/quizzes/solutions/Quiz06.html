Quiz 6

Statistical Significance, Errors, and Normal-Based Tests

Worked Solutions

Covers Sections 4.3, 4.4, 4.5, 5.1, 6.1

Instructions. This is a take-home quiz. You may use your lecture notes and online resources, but the work you submit must be your own. Show your work clearly to receive credit.

Total: 10 points

Problem 1 [2 pts]

For each p-value, decide whether to reject \(H_0\) at the given significance level. (0.5 pt each)

  1. p = 0.001, \(\alpha = 0.05\)
  2. p = 0.07, \(\alpha = 0.05\)
  3. p = 0.04, \(\alpha = 0.01\)
  4. p = 0.20, \(\alpha = 0.10\)

(a) Reject (p < 0.05). (b) Do not reject (p > 0.05). (c) Do not reject (p > 0.01). (d) Do not reject (p > 0.10).

Problem 2 [2 pts]

In a hypothesis test, define each in plain English: (1 pt each)

  1. Type I error
  2. Type II error

(a) Type I error: rejecting \(H_0\) when it is actually true (a false positive).

(b) Type II error: failing to reject \(H_0\) when it is actually false (a false negative).

Problem 3 [2 pts]

Compute the standardized test statistic.

  1. For a single proportion: \(\hat{p} = 0.62\), \(p_0 = 0.55\), \(n = 200\). Use \(z = (\hat{p} - p_0) / \sqrt{p_0(1-p_0)/n}\). (1 pt)
  2. For a single mean: \(\bar{x} = 102\), \(\mu_0 = 100\), \(s = 8\), \(n = 64\). Use \(z = (\bar{x} - \mu_0) / (s/\sqrt{n})\). (1 pt)

(a) \(SE = \sqrt{0.55 \cdot 0.45 / 200} = \sqrt{0.001238} \approx 0.0352\). \(z = (0.62 - 0.55)/0.0352 \approx 1.99\).

(b) \(SE = 8/\sqrt{64} = 1\). \(z = (102 - 100)/1 = 2\).

Problem 4 [2 pts]

The two-tailed p-value for \(z = 2.0\) is approximately 0.0455. State the conclusion at \(\alpha = 0.05\) for the test in Problem 3(b), and write a one-sentence interpretation. (1 pt each)

Since p ≈ 0.0455 < 0.05, reject \(H_0\). There is statistically significant evidence (at the 5% level) that the population mean differs from 100.

Problem 5 [2 pts]

The shaded region of the normal curve below shows the rejection region for a two-sided test at \(\alpha = 0.05\). Roughly what z-values mark the boundaries, and what would be the conclusion if the observed z-value is 1.5? (1 pt each)

Boundaries: \(z = \pm 1.96\). Since 1.5 is inside the (-1.96, 1.96) interval, we do not reject \(H_0\) at \(\alpha = 0.05\).
