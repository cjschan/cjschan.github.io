Problem 1 [2 pts]

In a hypothesis test, define each in plain English. (1 pt each)

  1. Type I error
  2. Type II error

Problem 2 [2 pts]

A 95% confidence interval for the mean is \((34.0,\ 39.0)\). Without doing a separate test, would you reject \(H_0: \mu = 36\) at \(\alpha = 0.05\)? Would you reject \(H_0: \mu = 40\)? Explain how the CI gives you this information.

Problem 3 [3 pts]

A large hospital study (\(n = 10{,}000\)) finds that a new medication lowers blood pressure by an average of 0.5 mmHg compared to a placebo, with a p-value of 0.003. (1 pt each)

  1. Is this result statistically significant at \(\alpha = 0.05\)?
  2. Do you think this result is practically significant? Explain.
  3. Explain how a very large sample size can lead to statistically significant results that are not practically meaningful.

Problem 4 [3 pts]

A 99% confidence interval for the difference in mean test scores between two teaching methods is \((-1.2,\ 8.4)\). (1 pt each)

  1. Based on this interval, would you reject \(H_0: \mu_1 - \mu_2 = 0\) at \(\alpha = 0.01\)? Explain.
  2. If we instead used \(\alpha = 0.05\), would the corresponding 95% confidence interval be wider or narrower?
  3. Describe a scenario in this context where a Type II error would occur, and explain its consequence.
