Section 4.4

Real Zeros of Polynomial Functions

Worked Solutions

How do we actually find the zeros of a higher-degree polynomial? We’ll use the factor theorem to connect zeros and linear factors, factoring techniques (common factor, grouping, substitution, special forms) to break a polynomial down, and graphs to estimate zeros when algebra doesn’t simplify cleanly.

Remainder and Factor Theorems

Factor Theorem

\((x - c)\) is a factor of \(f(x)\) if and only if \(f(c) = 0\). Equivalently:

  • If \(c\) is a zero of \(f\), then \((x - c)\) is a factor.
  • If \((x - c)\) is a factor, then \(c\) is a zero.
Using the Factor Theorem

The factor theorem lets you check whether a candidate value is a zero (just plug it in) and lets you build a polynomial when you know its zeros (multiply linear factors together).

Example 1

Verifying zeros

Decide whether each value is a zero of \(f(x) = x^3 - 2x^2 - 5x + 6\).

  1. \(x = 1\)
  2. \(x = 2\)

(a) \(f(1) = 1 - 2 - 5 + 6 = 0\) ✓. So \(x = 1\) is a zero, and \((x - 1)\) is a factor of \(f\).

(b) \(f(2) = 8 - 8 - 10 + 6 = -4 \neq 0\). So \(x = 2\) is not a zero.

Example 2

Building a polynomial from its zeros

Find a polynomial \(f\) of degree 3 with leading coefficient 1 whose zeros are \(2\), \(-1\), and \(3\).

Each zero gives a factor: \(f(x) = (x - 2)(x + 1)(x - 3)\).

Expand: \((x - 2)(x + 1) = x^2 - x - 2\). Multiply by \((x - 3)\): \(x^3 - 3x^2 - x^2 + 3x - 2x + 6 = x^3 - 4x^2 + x + 6\).

Check: each zero gives \(f = 0\) ✓.

Zeros by Factoring

Useful Factoring Techniques
  • Common factor: pull out any GCF first.
  • Grouping: for four-term polynomials, group two pairs and factor each pair to expose a common binomial factor.
  • Substitution: for polynomials of the form \(ax^4 + bx^2 + c\) (or other “quadratic in disguise”), let \(u = x^2\) and solve the resulting quadratic.
  • Special forms: difference of squares \(a^2 - b^2 = (a-b)(a+b)\); difference/sum of cubes \(a^3 \pm b^3 = (a \pm b)(a^2 \mp ab + b^2)\).
Example 3

Factoring by grouping

Find all real zeros of \(f(x) = x^3 + 2x^2 - 9x - 18\).

Group terms: \(x^3 + 2x^2 - 9x - 18 = x^2(x + 2) - 9(x + 2) = (x + 2)(x^2 - 9)\).

The second factor is a difference of squares: \(x^2 - 9 = (x - 3)(x + 3)\). So \(f(x) = (x + 2)(x - 3)(x + 3)\).

Zeros: \(x = -2, -3, 3\).

Example 4

Substitution

Find all real zeros of \(f(x) = x^4 - 5x^2 + 4\).

Let \(u = x^2\). Then \(f = u^2 - 5u + 4 = (u - 1)(u - 4)\).

Set each factor to zero: \(u = 1\) gives \(x^2 = 1\), so \(x = \pm 1\). \(u = 4\) gives \(x^2 = 4\), so \(x = \pm 2\).

Real zeros: \(\pm 1\) and \(\pm 2\).

Example 5

Common factor + difference of cubes

Find all real zeros of \(f(x) = 2x^4 - 16x\).

Pull out \(2x\): \(f(x) = 2x(x^3 - 8) = 2x(x - 2)(x^2 + 2x + 4)\) using the difference of cubes \(x^3 - 2^3\).

The quadratic factor has discriminant \(4 - 16 = -12 < 0\), so no real roots.

Real zeros: \(x = 0\) and \(x = 2\).

Approximating Zeros from a Graph

When Algebra Doesn’t Factor Cleanly

Not every polynomial factors over the integers. When it doesn’t, you can still:

  • Sketch the graph and look for sign changes — each sign change locates a real zero between consecutive integers.
  • Use a graphing calculator’s zero or root tool to approximate the value.
