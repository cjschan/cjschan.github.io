Section 2.2

Linear Equations

Worked Solutions

A linear equation is an equation that can be put into the form \(ax + b = 0\) with \(a \neq 0\). Today we develop a reliable strategy for solving them, including equations with parentheses, fractions, or decimals, and we apply the strategy to short word problems.

What is a Linear Equation?

Linear Equation in One Variable

A linear equation in the variable \(x\) is one that can be written as \(ax + b = 0\) with \(a \neq 0\). The solution is any value of \(x\) that makes both sides equal. A linear equation has exactly one solution.

Strategy for Solving Linear Equations
  1. Clear fractions or decimals (multiply by a common denominator or a power of 10).
  2. Use the distributive property to remove parentheses.
  3. Combine like terms on each side.
  4. Move all variable terms to one side, all constants to the other.
  5. Divide by the coefficient of the variable.
  6. Check by substituting back into the original equation.
Example 1

Multi-step linear equation

Solve.

  1. \(3(x - 2) - 5 = 2x + 4\)
  2. \(7 - 2(3x + 1) = 4x - 9\)

(a) Distribute: \(3x - 6 - 5 = 2x + 4\), i.e. \(3x - 11 = 2x + 4\). Subtract \(2x\): \(x - 11 = 4\). Add \(11\): \(\boxed{x = 15}\).

(b) Distribute: \(7 - 6x - 2 = 4x - 9\), i.e. \(5 - 6x = 4x - 9\). Add \(6x\) and \(9\): \(14 = 10x\), so \(\boxed{x = \tfrac{7}{5}}\).

Equations with Fractions

Example 2

Clearing fractions

Solve \(\dfrac{x}{3} + \dfrac{x - 1}{2} = \dfrac{5}{6}\).

Multiply both sides by the LCD, 6: \(6 \cdot \dfrac{x}{3} + 6 \cdot \dfrac{x-1}{2} = 6 \cdot \dfrac{5}{6}\), i.e. \(2x + 3(x - 1) = 5\). Distribute: \(2x + 3x - 3 = 5\), so \(5x = 8\), \(\boxed{x = \tfrac{8}{5}}\).

Example 3

Clearing decimals

Solve \(0.05x + 0.1(x - 4) = 1.7\).

Multiply both sides by 100 to clear decimals: \(5x + 10(x - 4) = 170\). Distribute: \(5x + 10x - 40 = 170\), so \(15x = 210\), \(\boxed{x = 14}\).

Identities and Contradictions

Three Possible Outcomes

When you solve a linear equation, exactly one of three things happens.

  • Conditional: ends in \(x = \text{number}\) — exactly one solution.
  • Identity: ends in a true statement like \(0 = 0\) — every real number is a solution.
  • Contradiction: ends in a false statement like \(2 = 5\) — no solution.
Example 4

Identity vs. contradiction

Solve.

  1. \(2(x + 3) = 2x + 6\)
  2. \(3(x - 1) + 5 = 3x + 1\)

(a) Distribute: \(2x + 6 = 2x + 6\). Subtract \(2x\) and \(6\): \(0 = 0\). Identity — every real number is a solution.

(b) Distribute: \(3x - 3 + 5 = 3x + 1\), i.e. \(3x + 2 = 3x + 1\). Subtract \(3x\): \(2 = 1\). Contradiction — no solution.

Linear Models

Example 5

A rental cost model

A truck rental costs $40 plus $0.55 per mile.

  1. Write a linear function \(C(m)\) for the cost of driving \(m\) miles.
  2. How many miles can you drive on a budget of $150?

(a) \(C(m) = 40 + 0.55 m\).

(b) Solve \(40 + 0.55 m = 150\): \(0.55 m = 110\), \(m = 200\) miles.

Example 6

Mixture problem

How many liters of a 50% acid solution must be added to 4 liters of a 20% acid solution to obtain a 30% acid solution?

Let \(x\) = liters of the 50% solution. Acid balance: \(0.50 x + 0.20(4) = 0.30(x + 4)\). So \(0.50 x + 0.80 = 0.30 x + 1.20\), giving \(0.20 x = 0.40\), \(x = 2\) liters.
