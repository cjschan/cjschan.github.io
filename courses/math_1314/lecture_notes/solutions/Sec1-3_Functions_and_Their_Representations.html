1.3 Functions and Their Representations

Basic Concepts and Function Notation

A function is a mathematical rule that associates a unique output value to each input value. This means that if you provide the same input to a function, you will always get the same output.

Example

Lightning causes thunder. However, we often see a flash of lightning before we hear the thunder because sound travels slower than light. The distance between a lightning strike and an observer affects the time delay between seeing the flash and hearing the thunder.

This calculation is represented by y = f(x) which is known as function notation. It is read as "y equals f of x." This notation indicates that the function f, when applied to the input x, produces the output y. In other words, f(Input) = Output.

f(x) = x/5

The function f takes the input and divides it by 5 to produce an output.

f(10) - what is the output when the input is 10?

y = f(x) = x/5

f(10) = 10/5 = 2

f(20) = 20/5 = 4

The variable y is called the dependent variable, and the variable x is called the independent variable.

A function computes exactly one output for each valid input.

Domain and Range of a Function

The set of all meaningful inputs x is called the domain of the function.

The set of corresponding outputs y is called the range of the function.

Set-Builder Notation

The expression {x | x > 1} is written in set-builder notation and represents the set of all real numbers x such that x is greater than 1.

Question: Is the number 2 in the set {x | x > 1}? Yes.

Question: What does the set {y | 1 < y < 5} represent?

The set of all y such that y is greater than 1 and y is less than 5

OR

The set of all y such that y is between 1 and 5, but does not include 1 or 5.

Is 6 in this set? No, because 6 is greater than 5. 6 is not strictly in between 1 and 5.

Set-builder and Interval notation are mathematical ways of writing a set of numbers.

Interval Notation: [2, ∞) "We use the bracket symbol if the number is included in the set"

Set notation: {x | x ≥ 2} "The set of all real numbers x such that x is greater than or equal to 2"

vs

(2, ∞) "This does not include 2"

"all real numbers from 2 to 5, but does not include 2 and includes 5"

0 1 2 3 4 5

Interval notation: (2, 5] "in this set, 2 is not included, but 5 is"

Set notation: {x | 2 < x ≤ 5} or {x | 2 < x and x ≤ 5} or {x | x > 2 and x ≤ 5}

Infinity is not a number, but we need to represent this in interval notation. So whenever you see an infinity symbol, you always need to use parentheses.

Identifying Functions: Vertical Line Test

If every vertical line intersects a graph at no more than one point, then the graph represents a function.
