Section 5.4

Logarithmic Functions and Models

Worked Solutions

The logarithm is the inverse of an exponential. Today we define logarithms, switch fluently between exponential and log form, use the basic properties to evaluate and rewrite expressions, and solve simple exponential and log equations.

What Is a Logarithm?

Logarithm

For \(a > 0\), \(a \neq 1\) and \(y > 0\), \[ \log_a y = x \quad \text{means} \quad a^x = y. \] In words, \(\log_a y\) is the exponent to which \(a\) must be raised to give \(y\). The base \(a\) cannot be 1, and the argument must be positive.

Common Bases
  • Common log: \(\log x = \log_{10} x\) (base 10).
  • Natural log: \(\ln x = \log_e x\) (base \(e\)).
Example 1

Switching between forms

Rewrite each in the other form.

  1. \(2^5 = 32\)
  2. \(\log_3 81 = 4\)
  3. \(e^x = 7\)
  4. \(\log y = 2.5\)

(a) \(\log_2 32 = 5\).

(b) \(3^4 = 81\).

(c) \(\ln 7 = x\), i.e. \(x = \ln 7\).

(d) \(10^{2.5} = y\).

Example 2

Evaluating logs by inspection

Evaluate without a calculator.

  1. \(\log_2 16\)
  2. \(\log_5 \dfrac{1}{25}\)
  3. \(\log_8 2\)
  4. \(\log 1000\)
  5. \(\ln 1\)

(a) \(2^4 = 16\), so \(\log_2 16 = 4\).

(b) \(5^{-2} = \tfrac{1}{25}\), so \(\log_5 \tfrac{1}{25} = -2\).

(c) Need \(8^x = 2\). Since \(8 = 2^3\), \(8^{1/3} = 2\), so \(\log_8 2 = \tfrac{1}{3}\).

(d) \(\log 1000 = 3\).

(e) \(\ln 1 = 0\) (any base of 1 is 0).

Properties of Logarithms

Log Rules

For positive \(M\), \(N\), real \(p\), and base \(a > 0\), \(a \neq 1\):

  • Product: \(\log_a (MN) = \log_a M + \log_a N\).
  • Quotient: \(\log_a \dfrac{M}{N} = \log_a M - \log_a N\).
  • Power: \(\log_a M^p = p \log_a M\).
  • Identity values: \(\log_a a = 1\), \(\log_a 1 = 0\).
  • Inverse: \(\log_a a^x = x\) and \(a^{\log_a x} = x\).
Example 3

Expanding and combining

Expand or combine as a single logarithm.

  1. Expand \(\log\dfrac{x^2 \sqrt{y}}{z^3}\).
  2. Combine \(2 \ln x - 3 \ln y + \ln z\) as a single \(\ln\).

(a) \(\log x^2 + \log y^{1/2} - \log z^3 = 2 \log x + \tfrac{1}{2} \log y - 3 \log z\).

(b) \(\ln x^2 - \ln y^3 + \ln z = \ln \dfrac{x^2 z}{y^3}\).

Change of Base

Change-of-Base Formula

For any positive \(M\), \[ \log_a M = \dfrac{\log_b M}{\log_b a} \] for any valid base \(b\). Calculators have buttons for base 10 (\(\log\)) and base \(e\) (\(\ln\)); use either one.

Example 4

Computing with change of base

Use a calculator to evaluate \(\log_3 50\) to four decimals.

\(\log_3 50 = \dfrac{\ln 50}{\ln 3} \approx \dfrac{3.9120}{1.0986} \approx 3.5609\).

Solving Exponential and Logarithmic Equations

Strategy
  • Exponential equation: isolate the exponential, then take \(\ln\) (or \(\log\)) of both sides.
  • Logarithmic equation: combine into a single log on one side, rewrite in exponential form, then solve.
  • Always check candidates against the original equation; logs of non-positive numbers are excluded.
Example 5

Solving an exponential equation

Solve \(3 \cdot 2^{x+1} = 96\). Round to three decimals if necessary.

Divide by 3: \(2^{x+1} = 32\). Note \(32 = 2^5\), so \(x + 1 = 5\), \(x = 4\). (Exact answer.)

Example 6

Solving a log equation

Solve \(\log_2 (x + 1) + \log_2 (x - 1) = 3\).

Combine: \(\log_2 [(x+1)(x-1)] = 3\), i.e. \(\log_2 (x^2 - 1) = 3\). Rewrite: \(x^2 - 1 = 2^3 = 8\). So \(x^2 = 9\), \(x = \pm 3\).

Check the domain: original logs require \(x > 1\). So \(x = 3\); reject \(-3\).

Logarithmic Models

Example 7

Doubling time

An investment grows continuously at 4.5% per year. How long does it take to double?

Solve \(P e^{0.045 t} = 2P\), i.e. \(e^{0.045 t} = 2\). Take \(\ln\): \(0.045 t = \ln 2\), so \(t = \dfrac{\ln 2}{0.045} \approx 15.40\) years.
