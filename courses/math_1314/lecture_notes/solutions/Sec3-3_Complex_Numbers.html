Section 3.3

Complex Numbers

Worked Solutions · View student notes →

The equation \(x^2 = -1\) has no real solution because real squares are never negative. Mathematicians invented a number \(i\) with \(i^2 = -1\). This extension — the complex numbers — lets every quadratic have a solution.

The Imaginary Unit and Complex Numbers

The Imaginary Unit

The imaginary unit \(i\) is defined by \(i^2 = -1\). For any positive real \(b\), \[ \sqrt{-b} = i\sqrt{b}. \]

Complex Number

A complex number is one of the form \(a + bi\), where \(a\) and \(b\) are real numbers. The real part is \(a\) and the imaginary part is \(b\). When \(b = 0\) we have a real number; when \(a = 0\) and \(b \neq 0\) we have a pure imaginary.

Example 1

Square roots of negatives

Simplify.

  1. \(\sqrt{-25}\)
  2. \(\sqrt{-72}\)
  3. \(-\sqrt{-9}\)

(a) \(\sqrt{-25} = i\sqrt{25} = 5i\).

(b) \(\sqrt{-72} = i\sqrt{72} = i\sqrt{36 \cdot 2} = 6i\sqrt{2}\).

(c) \(-\sqrt{-9} = -3i\).

Arithmetic of Complex Numbers

Adding, Subtracting, Multiplying

Treat \(i\) as a variable, then replace \(i^2\) with \(-1\).

  • \((a + bi) + (c + di) = (a + c) + (b + d)i\)
  • \((a + bi) - (c + di) = (a - c) + (b - d)i\)
  • \((a + bi)(c + di) = ac + adi + bci + bdi^2 = (ac - bd) + (ad + bc)i\)
Example 2

Add, subtract, multiply

Simplify.

  1. \((3 + 2i) + (4 - 5i)\)
  2. \((6 - i) - (2 + 4i)\)
  3. \((3 + 2i)(4 - i)\)
  4. \((1 + 3i)^2\)

(a) \((3 + 4) + (2 - 5)i = 7 - 3i\).

(b) \((6 - 2) + (-1 - 4)i = 4 - 5i\).

(c) \(12 - 3i + 8i - 2i^2 = 12 + 5i + 2 = 14 + 5i\).

(d) \((1 + 3i)^2 = 1 + 6i + 9i^2 = 1 + 6i - 9 = -8 + 6i\).

Complex Conjugates and Division

Complex Conjugate

The complex conjugate of \(a + bi\) is \(\overline{a + bi} = a - bi\). Conjugates multiply to a real number: \((a + bi)(a - bi) = a^2 + b^2\).

Example 3

Dividing complex numbers

Write each quotient in the form \(a + bi\).

  1. \(\dfrac{3 + i}{2 - i}\)
  2. \(\dfrac{5}{1 + 2i}\)

(a) Multiply top and bottom by \(2 + i\): \(\dfrac{(3+i)(2+i)}{(2-i)(2+i)} = \dfrac{6 + 3i + 2i + i^2}{4 + 1} = \dfrac{5 + 5i}{5} = 1 + i\).

(b) Multiply by \(1 - 2i\): \(\dfrac{5(1 - 2i)}{(1+2i)(1-2i)} = \dfrac{5 - 10i}{1 + 4} = \dfrac{5 - 10i}{5} = 1 - 2i\).

Quadratic Equations with Complex Solutions

Example 4

Negative discriminant

Solve.

  1. \(x^2 + 16 = 0\)
  2. \(x^2 - 4x + 13 = 0\)
  3. \(2x^2 + 3x + 5 = 0\)

(a) \(x^2 = -16\), so \(x = \pm 4i\).

(b) Quadratic formula: \(x = \dfrac{4 \pm \sqrt{16 - 52}}{2} = \dfrac{4 \pm \sqrt{-36}}{2} = \dfrac{4 \pm 6i}{2} = 2 \pm 3i\).

(c) \(D = 9 - 40 = -31\). \(x = \dfrac{-3 \pm \sqrt{-31}}{4} = -\dfrac{3}{4} \pm \dfrac{\sqrt{31}}{4} i\).

Powers of \(i\)

Cycle of Powers

Powers of \(i\) repeat in a cycle of 4: \[ i^1 = i, \quad i^2 = -1, \quad i^3 = -i, \quad i^4 = 1, \quad i^5 = i, \ldots \] To compute \(i^n\), divide \(n\) by 4 and use the remainder.

Example 5

High powers of \(i\)

Simplify.

  1. \(i^{17}\)
  2. \(i^{30}\)
  3. \(i^{-3}\)

(a) \(17 \div 4\) gives remainder 1, so \(i^{17} = i\).

(b) \(30 \div 4\) gives remainder 2, so \(i^{30} = -1\).

(c) \(i^{-3} = \dfrac{1}{i^3} = \dfrac{1}{-i} = \dfrac{1}{-i} \cdot \dfrac{i}{i} = \dfrac{i}{-i^2} = \dfrac{i}{1} = i\).
