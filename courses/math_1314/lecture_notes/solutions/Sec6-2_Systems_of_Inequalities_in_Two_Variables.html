Section 6.2

Systems of Inequalities in Two Variables

Worked Solutions · View student notes →

A linear inequality in two variables describes a half-plane. A system of such inequalities describes the intersection — a region of the plane — whose points satisfy every inequality. Today we graph these regions and use them in simple optimization.

Graphing a Single Linear Inequality

Strategy
  1. Replace the inequality with equality and graph the boundary line.
  2. Use a solid line for \(\leq\) or \(\geq\) (line is included). Use a dashed line for \(<\) or \(>\) (line excluded).
  3. Pick a test point not on the line (such as \((0, 0)\) when possible) and check whether it satisfies the original inequality. If yes, shade the side of the line containing the test point; otherwise shade the other side.
Example 1

A single inequality

Graph \(2x - y \leq 4\).

Boundary: \(2x - y = 4\), i.e. \(y = 2x - 4\) — solid line through \((0, -4)\) and \((2, 0)\). Test \((0, 0)\): \(2(0) - 0 = 0 \leq 4\) ✓. Shade the side of the line containing the origin (above the line).

Systems of Inequalities

Solution of a System

The solution of a system of inequalities in two variables is the set of points \((x, y)\) that satisfy every inequality. It is the intersection of the individual solution regions.

Example 2

Two-inequality system

Graph the solution.

\(\begin{cases} y \geq -x + 2 \\ y < 2x - 1 \end{cases}\)

Solid line \(y = -x + 2\); shade above (test \((3, 0)\): \(0 \geq -1\) ✓).

Dashed line \(y = 2x - 1\); shade below (test \((0, 0)\): \(0 < -1\) ✗, so shade the opposite side, which is below the line for points like \((3, 0)\): \(0 < 5\) ✓).

The solution is the wedge above the first line and below the second — an unbounded region.

Example 3

Bounded region

Graph the solution.

\(\begin{cases} x + y \leq 6 \\ x \geq 0 \\ y \geq 0 \end{cases}\)

The first inequality cuts off the half-plane below the line \(x + y = 6\). The other two restrict to the first quadrant. The intersection is a triangle with vertices \((0, 0)\), \((6, 0)\), and \((0, 6)\), including the boundary.

Reading Vertices (Corner Points)

Corner Points

The corner points of a feasible region are the intersections of the boundary lines that lie within the region. They are crucial in linear programming and other optimization tasks.

Example 4

Identify corners

Find the corner points of the region defined by \(\begin{cases} 2x + y \leq 8 \\ x + 2y \leq 10 \\ x \geq 0 \\ y \geq 0 \end{cases}\).

Solve pairwise intersections:

  • \(x = 0, y = 0 \Rightarrow (0, 0)\) ✓
  • \(y = 0\) and \(2x + y = 8 \Rightarrow (4, 0)\) ✓
  • \(x = 0\) and \(x + 2y = 10 \Rightarrow (0, 5)\) ✓
  • \(2x + y = 8\) and \(x + 2y = 10\): multiply the first by 2 and subtract: \(4x + 2y - x - 2y = 16 - 10\), \(3x = 6\), \(x = 2\). Then \(y = 4\). Corner \((2, 4)\) ✓

Corners: \((0, 0)\), \((4, 0)\), \((2, 4)\), \((0, 5)\).

An Application: Optimization

Linear Programming Theorem

If a linear function (the objective) attains a maximum or minimum on a closed bounded region defined by linear inequalities, that optimum occurs at a corner point of the region.

Example 5

Maximize a linear objective

Using the region from Example 4, find the maximum and minimum values of \(P = 3x + 5y\) on the region.

Evaluate \(P\) at each corner.

  • \((0, 0)\): \(P = 0\)
  • \((4, 0)\): \(P = 12\)
  • \((2, 4)\): \(P = 6 + 20 = 26\) ← max
  • \((0, 5)\): \(P = 25\)

Maximum 26 at \((2, 4)\); minimum 0 at \((0, 0)\).
