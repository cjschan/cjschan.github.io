Section 5.1

Combining Functions

Worked Solutions

Functions can be added, subtracted, multiplied, and divided just like numbers, and they can also be composed — using the output of one function as the input of another. Each operation has its own domain rules.

Arithmetic Combinations

Sum, Difference, Product, Quotient

For functions \(f\) and \(g\):

  • \((f + g)(x) = f(x) + g(x)\)
  • \((f - g)(x) = f(x) - g(x)\)
  • \((fg)(x) = f(x) \, g(x)\)
  • \(\left(\dfrac{f}{g}\right)(x) = \dfrac{f(x)}{g(x)}\), provided \(g(x) \neq 0\)

The domain of each combination is the intersection of the domains of \(f\) and \(g\), excluding values where the denominator is zero (for the quotient).

Example 1

Computing combinations

Let \(f(x) = x^2 - 4\) and \(g(x) = x + 2\).

  1. Find \((f + g)(3)\).
  2. Find \((fg)(x)\) and simplify.
  3. Find \(\left(\dfrac{f}{g}\right)(x)\) and simplify, stating any domain restrictions.

(a) \(f(3) = 5\), \(g(3) = 5\). \((f + g)(3) = 10\).

(b) \((fg)(x) = (x^2 - 4)(x + 2) = (x - 2)(x + 2)(x + 2) = (x - 2)(x + 2)^2\).

(c) \(\dfrac{f(x)}{g(x)} = \dfrac{x^2 - 4}{x + 2} = \dfrac{(x - 2)(x + 2)}{x + 2} = x - 2\), with \(x \neq -2\).

Example 2

Domain of a quotient

Let \(f(x) = \sqrt{x + 1}\) and \(g(x) = x - 3\). Find the domain of \(\dfrac{f}{g}\).

Domain of \(f\): \(x + 1 \geq 0\), i.e. \(x \geq -1\). Domain of \(g\): all reals.

For the quotient, also exclude \(x = 3\) (where \(g\) is zero). Domain: \([-1, 3) \cup (3, \infty)\).

Composition of Functions

Composition

The composition of \(f\) and \(g\) is \((f \circ g)(x) = f(g(x))\) — first apply \(g\), then feed the result into \(f\). The domain of \(f \circ g\) consists of those \(x\) in the domain of \(g\) such that \(g(x)\) lies in the domain of \(f\).

Example 3

Compute compositions

Let \(f(x) = 2x - 5\) and \(g(x) = x^2 + 1\). Find each.

  1. \((f \circ g)(3)\)
  2. \((g \circ f)(3)\)
  3. \((f \circ g)(x)\)
  4. \((g \circ f)(x)\)

(a) \(g(3) = 10\); \(f(10) = 15\).

(b) \(f(3) = 1\); \(g(1) = 2\).

(c) \(f(g(x)) = 2(x^2 + 1) - 5 = 2x^2 - 3\).

(d) \(g(f(x)) = (2x - 5)^2 + 1 = 4x^2 - 20x + 26\).

Example 4

Domain of a composition

Let \(f(x) = \sqrt{x}\) and \(g(x) = x - 4\). Find \((f \circ g)(x)\) and its domain.

\((f \circ g)(x) = f(x - 4) = \sqrt{x - 4}\). Need \(x - 4 \geq 0\), so the domain is \([4, \infty)\).

Decomposing Functions

Decomposition

To express \(h\) as a composition \(f \circ g\), look for an “inner” function (often the expression inside parentheses, under a radical, or in an exponent) and an “outer” function (everything done to that inner expression).

Example 5

Inside and outside

Express each function as a composition \(f(g(x))\).

  1. \(h(x) = (3x + 1)^7\)
  2. \(h(x) = \sqrt{x^2 + 5}\)
  3. \(h(x) = \dfrac{1}{(2x - 3)^2}\)

(a) Inner \(g(x) = 3x + 1\), outer \(f(u) = u^7\).

(b) Inner \(g(x) = x^2 + 5\), outer \(f(u) = \sqrt{u}\).

(c) Inner \(g(x) = 2x - 3\), outer \(f(u) = \dfrac{1}{u^2}\).

An Application

Example 6

Two functions chained

A factory’s daily revenue (in thousands of dollars) when \(x\) hundred items are sold is \(R(x) = 4x\). The number of items sold depends on the price \(p\) (dollars) by \(x(p) = 50 - 2p\). Find revenue as a function of price and use it to find the price that produces revenue of \$120{,}000\).

Compose: \(R(x(p)) = 4(50 - 2p) = 200 - 8p\) (thousand dollars).

Solve \(200 - 8p = 120\): \(8p = 80\), so \(p = \$10\).
