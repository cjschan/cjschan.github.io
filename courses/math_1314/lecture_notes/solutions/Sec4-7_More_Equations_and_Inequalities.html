Section 4.7

More Equations and Inequalities

Worked Solutions

We extend the equation- and inequality-solving toolkit to rational and radical equations, then use sign analysis to handle rational inequalities.

Rational Equations

Solving Rational Equations
  1. Find the LCD of all fractions.
  2. Multiply both sides of the equation by the LCD to clear fractions.
  3. Solve the resulting polynomial equation.
  4. Check each candidate against the original equation; reject any value that makes a denominator zero (these are extraneous).
Example 1

Clearing fractions

Solve \(\dfrac{2}{x} + \dfrac{1}{x + 3} = \dfrac{8}{3}\).

LCD: \(3x(x + 3)\). Multiply: \(3x(x+3) \cdot \dfrac{2}{x} + 3x(x+3) \cdot \dfrac{1}{x+3} = 3x(x+3) \cdot \dfrac{8}{3}\).

Simplify: \(6(x + 3) + 3x = 8x(x + 3)\), i.e. \(6x + 18 + 3x = 8x^2 + 24x\), so \(8x^2 + 15x - 18 = 0\).

Quadratic formula: \(x = \dfrac{-15 \pm \sqrt{225 + 576}}{16} = \dfrac{-15 \pm \sqrt{801}}{16}\). Both candidates are non-zero and not \(-3\), so both are valid.

Example 2

Watch for extraneous roots

Solve \(\dfrac{x}{x - 2} = \dfrac{2}{x - 2} + 4\).

Multiply by \(x - 2\): \(x = 2 + 4(x - 2)\), so \(x = 2 + 4x - 8\), giving \(-3x = -6\), \(x = 2\).

But \(x = 2\) makes the original denominator zero, so it is extraneous. There is no solution.

Radical Equations

Solving Radical Equations
  1. Isolate one radical on one side of the equation.
  2. Raise both sides to the appropriate power to remove the radical.
  3. If a radical remains, isolate and raise again.
  4. Solve the resulting equation.
  5. Check each candidate — squaring can introduce extraneous solutions.
Example 3

Single radical

Solve \(\sqrt{2x + 5} = x + 1\).

Square both sides: \(2x + 5 = (x + 1)^2 = x^2 + 2x + 1\). Simplify: \(5 = x^2 + 1\), so \(x^2 = 4\), \(x = \pm 2\).

Check \(x = 2\): \(\sqrt{9} = 3\) and \(2 + 1 = 3\) ✓. Check \(x = -2\): \(\sqrt{1} = 1\) but \(-2 + 1 = -1\) ✗ (extraneous).

Solution: \(x = 2\).

Example 4

Two radicals

Solve \(\sqrt{x + 7} - \sqrt{x} = 1\).

Isolate: \(\sqrt{x + 7} = 1 + \sqrt{x}\). Square: \(x + 7 = 1 + 2\sqrt{x} + x\), so \(6 = 2\sqrt{x}\), \(\sqrt{x} = 3\), \(x = 9\).

Check: \(\sqrt{16} - \sqrt{9} = 4 - 3 = 1\) ✓.

Rational Inequalities

Sign-Chart Method
  1. Write the inequality with one side equal to 0.
  2. Combine into a single fraction \(\dfrac{p(x)}{q(x)}\).
  3. Find the values that make the numerator or denominator zero. These divide the number line into intervals.
  4. Test one value from each interval to find the sign.
  5. Read the solution; exclude values that make the denominator 0 even when the inequality is non-strict.
Example 5

Rational inequality

Solve \(\dfrac{x - 1}{x + 2} \geq 0\). Write the answer in interval notation.

Boundaries: numerator zero at \(x = 1\); denominator zero at \(x = -2\) (excluded).

Test \(x = -3\): \(\dfrac{-4}{-1} = 4 > 0\). Test \(x = 0\): \(\dfrac{-1}{2} < 0\). Test \(x = 2\): \(\dfrac{1}{4} > 0\).

Solution: \((-\infty, -2) \cup [1, \infty)\). Note the bracket at \(1\) (numerator can be 0) and the open at \(-2\) (denominator cannot).

Example 6

Inequality with a quadratic numerator

Solve \(\dfrac{x^2 - 4}{x - 1} \leq 0\).

Numerator: \((x - 2)(x + 2) = 0\) at \(x = \pm 2\). Denominator: \(x = 1\) (excluded).

Boundaries on the number line: \(-2, 1, 2\). Test points:

  • \(x = -3\): \(\dfrac{5}{-4} < 0\) ✓
  • \(x = 0\): \(\dfrac{-4}{-1} = 4 > 0\) ✗
  • \(x = 1.5\): \(\dfrac{-1.75}{0.5} < 0\) ✓
  • \(x = 3\): \(\dfrac{5}{2} > 0\) ✗

Include \(\pm 2\) (numerator zero, expression equals 0). Exclude \(1\). Solution: \((-\infty, -2] \cup (1, 2]\).
