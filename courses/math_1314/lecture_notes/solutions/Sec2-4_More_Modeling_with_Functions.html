Section 2.4

More Modeling with Functions

Worked Solutions

Many real-world relationships can be modeled by linear functions or by simple proportions called direct variation. Today we set up function models from word problems and learn to read piecewise-defined functions.

Direct Variation

Direct Variation

The variable \(y\) varies directly with \(x\) (or is directly proportional to \(x\)) if there is a constant \(k\) such that \(y = kx\). The number \(k\) is the constant of variation.

Example 1

Setting up a direct variation

The cost of beef varies directly with weight. If 3 pounds cost $13.50, find the constant of variation and the cost of 5.5 pounds.

Let \(C = kw\). Plug in \(C = 13.50\) and \(w = 3\): \(13.50 = 3k\), so \(k = 4.50\) dollars per pound. For 5.5 pounds, \(C = 4.50(5.5) = \$24.75\).

Example 2

Hooke’s law

The distance a spring stretches is directly proportional to the weight hung from it. A 12-lb weight stretches the spring 9 in.

  1. Find the constant of variation.
  2. How far will the spring stretch under a 20-lb weight?

(a) Let \(d = kw\). Then \(9 = k(12)\), so \(k = \tfrac{9}{12} = 0.75\) in/lb.

(b) \(d = 0.75(20) = 15\) in.

Linear Models from Two Data Points

Example 3

From data to a model

A printing service charges a fixed setup fee plus a per-page price. 50 pages cost $40, and 200 pages cost $115.

  1. Find a linear model \(C(p)\) for the cost of printing \(p\) pages.
  2. What is the setup fee?
  3. What is the cost per page?
  4. What is the cost for 350 pages?

(a) Slope: \(m = \dfrac{115 - 40}{200 - 50} = \dfrac{75}{150} = 0.5\). Use \((50, 40)\): \(40 = 0.5(50) + b\), so \(b = 15\). Model: \(C(p) = 0.5 p + 15\).

(b) Setup fee: \(C(0) = \$15\).

(c) Cost per page: \$0.50.

(d) \(C(350) = 0.5(350) + 15 = \$190\).

Piecewise-Defined Functions

Piecewise-Defined Function

A piecewise-defined function uses different formulas on different parts of its domain. To evaluate or graph it, first decide which piece applies to the input.

Example 4

Evaluating a piecewise function

Let \(\displaystyle f(x) = \begin{cases} 2x + 1 & \text{if } x < 0 \\ x^2 & \text{if } 0 \leq x \leq 3 \\ -x + 12 & \text{if } x > 3 \end{cases}\)

Evaluate \(f(-2)\), \(f(0)\), \(f(2)\), \(f(3)\), and \(f(5)\).

\(f(-2)\): \(-2 < 0\), use \(2x + 1\). \(f(-2) = 2(-2) + 1 = -3\).

\(f(0)\): \(0\) satisfies \(0 \leq x \leq 3\), use \(x^2\). \(f(0) = 0\).

\(f(2)\): use \(x^2\). \(f(2) = 4\).

\(f(3)\): use \(x^2\). \(f(3) = 9\).

\(f(5)\): \(5 > 3\), use \(-x + 12\). \(f(5) = 7\).

Example 5

Cell-phone overage

A cell-phone plan includes 4 GB of data for $35 per month. Each additional GB beyond 4 costs $10. Write a piecewise function \(C(g)\) for the monthly cost, where \(g \geq 0\) is the gigabytes used in the month. Then compute \(C(2)\) and \(C(7)\).

If \(0 \leq g \leq 4\), the cost is just \$35. If \(g > 4\), the user pays \$35 plus \$10 for each GB beyond 4: \(35 + 10(g - 4)\).

\(\displaystyle C(g) = \begin{cases} 35 & \text{if } 0 \leq g \leq 4 \\ 35 + 10(g - 4) & \text{if } g > 4 \end{cases}\)

\(C(2) = \$35\). \(C(7) = 35 + 10(3) = \$65\).
