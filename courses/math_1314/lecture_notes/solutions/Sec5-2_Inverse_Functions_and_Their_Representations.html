Section 5.2

Inverse Functions and Their Representations

Worked Solutions

An inverse function undoes what the original does. Today we test whether a function has an inverse (one-to-one), find inverses algebraically, graphically, and numerically, and verify with composition.

One-to-One Functions

One-to-One

A function \(f\) is one-to-one if every output corresponds to exactly one input: \(f(a) = f(b)\) implies \(a = b\). Only one-to-one functions have inverses.

Horizontal Line Test

A function is one-to-one if and only if every horizontal line crosses its graph at most once.

Example 1

Horizontal line test

Decide whether each function is one-to-one.

  1. \(f(x) = 3x + 7\)
  2. \(g(x) = x^2\)
  3. \(h(x) = x^3\)
  4. \(k(x) = |x|\)

(a) Linear with non-zero slope — one-to-one.

(b) Parabola: e.g., \(g(2) = g(-2)\). Not one-to-one.

(c) Cubic, strictly increasing — one-to-one.

(d) V-shape: \(k(2) = k(-2)\). Not one-to-one.

Inverse Functions

Inverse Function

If \(f\) is one-to-one, the inverse function \(f^{-1}\) is defined by \[ f^{-1}(y) = x \quad \text{precisely when} \quad f(x) = y. \] In words: \(f^{-1}\) reverses the action of \(f\). The inputs of \(f^{-1}\) are the outputs of \(f\).

Note: \(f^{-1}\) does not mean \(\frac{1}{f}\).

Inverse Properties
  • Domain of \(f^{-1}\) = Range of \(f\).
  • Range of \(f^{-1}\) = Domain of \(f\).
  • \(f^{-1}(f(x)) = x\) for every \(x\) in the domain of \(f\).
  • \(f(f^{-1}(y)) = y\) for every \(y\) in the domain of \(f^{-1}\).
  • The graph of \(f^{-1}\) is the reflection of the graph of \(f\) across the line \(y = x\).

Finding an Inverse Algebraically

Strategy
  1. Replace \(f(x)\) with \(y\).
  2. Swap \(x\) and \(y\).
  3. Solve for \(y\).
  4. Replace \(y\) with \(f^{-1}(x)\).
Example 2

Linear inverse

Find the inverse of \(f(x) = \dfrac{2x - 1}{3}\). Verify by composition.

Set \(y = \dfrac{2x - 1}{3}\). Swap: \(x = \dfrac{2y - 1}{3}\). Solve: \(3x = 2y - 1\), \(2y = 3x + 1\), \(y = \dfrac{3x + 1}{2}\). So \(f^{-1}(x) = \dfrac{3x + 1}{2}\).

Verify: \(f(f^{-1}(x)) = \dfrac{2 \cdot \frac{3x+1}{2} - 1}{3} = \dfrac{3x + 1 - 1}{3} = x\) ✓.

Example 3

Restricting the domain

The function \(f(x) = x^2\) is not one-to-one. Restrict the domain to \(x \geq 0\) and find the inverse on that domain.

On \([0, \infty)\), \(f\) is one-to-one. Setting \(y = x^2\) and swapping: \(x = y^2\); solving for \(y \geq 0\) gives \(y = \sqrt{x}\). So \(f^{-1}(x) = \sqrt{x}\) with domain \([0, \infty)\).

Graphs of Inverse Functions

Example 4

Reflection across \(y = x\)

Let \(f(x) = 2x + 1\). Find \(f^{-1}\) and graph both functions on the same axes.

Setting \(y = 2x + 1\) and swapping: \(x = 2y + 1\), so \(y = \dfrac{x - 1}{2}\). Hence \(f^{-1}(x) = \dfrac{x - 1}{2}\).

The two graphs are reflections of each other across the line \(y = x\) (shown dashed).

Inverses from Tables

Example 5

Numerical inverse

The table represents a one-to-one function \(f\):

\(x\)1234
\(f(x)\)591120

Find \(f^{-1}(11)\) and \(f^{-1}(5)\). What does the table for \(f^{-1}\) look like?

\(f^{-1}(11) = 3\) (because \(f(3) = 11\)). \(f^{-1}(5) = 1\). Swap the rows to get the table for \(f^{-1}\):

\(x\)591120
\(f^{-1}(x)\)1234
