Section 3.5

Transformation of Graphs

Worked Solutions · View student notes →

You can build complicated graphs from a few parent functions using rigid shifts, reflections, and stretches. Recognizing transformations lets you sketch a graph quickly without plotting points.

Parent Functions

Common Parent Functions
  • Linear: \(y = x\)
  • Quadratic: \(y = x^2\)
  • Cubic: \(y = x^3\)
  • Square root: \(y = \sqrt{x}\)
  • Reciprocal: \(y = \dfrac{1}{x}\)
  • Reciprocal square: \(y = \dfrac{1}{x^2}\)
  • Absolute value: \(y = |x|\)

Vertical and Horizontal Shifts

Shifts

Let \(c > 0\). Compared to \(y = f(x)\):

  • \(y = f(x) + c\) shifts the graph up by \(c\).
  • \(y = f(x) - c\) shifts the graph down by \(c\).
  • \(y = f(x - c)\) shifts the graph right by \(c\).
  • \(y = f(x + c)\) shifts the graph left by \(c\).

Horizontal shifts go in the opposite direction of the sign you see — \(f(x - 3)\) shifts right.

Example 1

Identifying shifts

Describe each as a transformation of the parent graph \(y = x^2\).

  1. \(y = x^2 + 4\)
  2. \(y = (x - 5)^2\)
  3. \(y = (x + 2)^2 - 3\)

(a) Shift up 4.

(b) Shift right 5.

(c) Shift left 2 and down 3. Vertex at \((-2, -3)\).

Reflections

Reflections
  • \(y = -f(x)\) reflects across the \(x\)-axis (flip top-to-bottom).
  • \(y = f(-x)\) reflects across the \(y\)-axis (flip left-to-right).
Example 2

Reflections

Describe how to obtain each graph from \(y = \sqrt{x}\).

  1. \(y = -\sqrt{x}\)
  2. \(y = \sqrt{-x}\)
  3. \(y = -\sqrt{-x}\)

(a) Reflect \(\sqrt{x}\) across the \(x\)-axis. The graph lies in the fourth quadrant.

(b) Reflect across the \(y\)-axis. The graph lies in the second quadrant (domain becomes \(x \leq 0\)).

(c) Reflect across both axes. The graph lies in the third quadrant.

Vertical and Horizontal Stretches/Compressions

Stretches and Compressions

Let \(a > 0\).

  • \(y = a f(x)\) is a vertical stretch by factor \(a\) (taller) if \(a > 1\), or a vertical compression if \(0 < a < 1\).
  • \(y = f(ax)\) is a horizontal compression by factor \(a\) if \(a > 1\), or a horizontal stretch if \(0 < a < 1\).
Example 3

Combined transformations

Describe the transformations from \(y = x^2\) to \(y = -2(x + 1)^2 + 5\), in order.

Read the formula in the order (i) horizontal shift, (ii) horizontal stretch/compression, (iii) reflection, (iv) vertical stretch/compression, (v) vertical shift.

  1. Shift left 1: \(y = (x + 1)^2\).
  2. Vertical stretch by factor 2: \(y = 2(x + 1)^2\).
  3. Reflect across the \(x\)-axis: \(y = -2(x + 1)^2\).
  4. Shift up 5: \(y = -2(x + 1)^2 + 5\).

Vertex at \((-1, 5)\), opens down, narrower than \(x^2\).

Example 4

Sketching with transformations

Sketch \(g(x) = -|x - 3| + 4\) by transforming the graph of \(f(x) = |x|\). Identify the vertex.

Start with \(y = |x|\) (V-shape, vertex at origin). Shift right 3 to get \(|x - 3|\). Reflect across the \(x\)-axis to get \(-|x - 3|\) (V flipped to ∧). Shift up 4 to get the final graph. Vertex: \((3, 4)\); the graph opens downward.

Order Matters

Suggested Order

When several transformations are applied, perform horizontal changes (shift, stretch, reflect about the \(y\)-axis) inside the function, then vertical changes outside, in this order:

  1. Horizontal shift
  2. Horizontal stretch/compression
  3. Reflection across the \(y\)-axis
  4. Vertical stretch/compression
  5. Reflection across the \(x\)-axis
  6. Vertical shift
Example 5

Find the formula

The graph of \(y = \sqrt{x}\) is shifted right 4 units, reflected across the \(x\)-axis, vertically stretched by a factor of 3, then shifted up 2 units. Write a formula for the resulting function.

Step by step: \(\sqrt{x} \to \sqrt{x - 4} \to -\sqrt{x - 4} \to -3\sqrt{x - 4} \to -3\sqrt{x - 4} + 2\).

Final formula: \(y = -3\sqrt{x - 4} + 2\).
