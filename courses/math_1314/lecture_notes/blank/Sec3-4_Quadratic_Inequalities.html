Section 3.4

Quadratic Inequalities

Student Notes · View worked solutions →

A quadratic inequality has the form \(ax^2 + bx + c\) with one of \(<, \leq, >, \geq\) zero. We solve them graphically (by reading where the parabola is above or below the \(x\)-axis) or algebraically (using a sign chart).

Reading Inequalities from a Graph

Graphical Strategy
  1. Solve \(ax^2 + bx + c = 0\) to find the \(x\)-intercepts of the parabola.
  2. Sketch the parabola, opening up if \(a > 0\) or down if \(a < 0\).
  3. The parabola is above the \(x\)-axis where its values are positive, and below where they are negative.
  4. Read the solution off the graph; use brackets when the inequality includes equality.
Example 1

Solving by graph

Use the graph of \(f(x) = x^2 - 4x + 3\) to solve each inequality.

  1. \(x^2 - 4x + 3 < 0\)
  2. \(x^2 - 4x + 3 \geq 0\)

Sign-Chart Method

Sign-Chart Strategy
  1. Solve \(ax^2 + bx + c = 0\) to find the boundary values.
  2. Mark them on a number line, dividing it into open intervals.
  3. Pick one test value from each interval and plug it into \(ax^2 + bx + c\). Record the resulting sign.
  4. The solution is the union of intervals where the sign matches the inequality.
Example 2

Two-step quadratic inequality

Solve \(2x^2 - 5x - 3 \geq 0\). Write your answer in interval notation.
