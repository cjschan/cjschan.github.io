Section 4.6

Rational Functions and Models

Student Notes

A rational function is a quotient of two polynomials. Its graph can have vertical asymptotes (where the denominator is zero), horizontal asymptotes (controlling end behavior), and removable holes. We’ll find these features and use them to sketch.

Rational Functions

Rational Function

A rational function is a function of the form \(f(x) = \dfrac{p(x)}{q(x)}\), where \(p\) and \(q\) are polynomials and \(q\) is not the zero polynomial. The domain consists of all real \(x\) with \(q(x) \neq 0\).

Example 1

Domain

Find the domain of each rational function.

  1. \(f(x) = \dfrac{x + 1}{x - 4}\)
  2. \(g(x) = \dfrac{x}{x^2 - 9}\)
  3. \(h(x) = \dfrac{2x}{x^2 + 1}\)

Vertical Asymptotes and Holes

Vertical Asymptotes vs. Holes

Suppose \(f(x) = \dfrac{p(x)}{q(x)}\) and \(q(c) = 0\).

  • If the factor \((x - c)\) cancels in lowest terms, the graph has a hole at \(x = c\).
  • If it does not cancel, the line \(x = c\) is a vertical asymptote.
Example 2

Asymptote or hole?

Find the vertical asymptotes and holes of each.

  1. \(f(x) = \dfrac{x + 2}{x^2 - 4}\)
  2. \(g(x) = \dfrac{x^2 - 1}{x^2 - 5x + 4}\)

Horizontal Asymptotes

Horizontal Asymptote Rule

Compare the degrees of the numerator (\(n\)) and denominator (\(m\)).

  • \(n < m\): horizontal asymptote \(y = 0\).
  • \(n = m\): horizontal asymptote \(y = \dfrac{a_n}{b_m}\) (ratio of leading coefficients).
  • \(n > m\): no horizontal asymptote (end behavior follows the polynomial division).
Example 3

Horizontal asymptotes

Find any horizontal asymptotes.

  1. \(f(x) = \dfrac{2}{x + 5}\)
  2. \(g(x) = \dfrac{3x^2 - 1}{x^2 + 4}\)
  3. \(h(x) = \dfrac{x^3}{x^2 - 1}\)

Rational Models

Example 6

Average cost

A small printer’s daily fixed costs are $300, and producing each book costs $4. The average cost per book when \(x\) books are produced is

\[ A(x) = \frac{300 + 4x}{x}, \quad x > 0. \]

  1. What is the horizontal asymptote? Interpret it.
  2. Compute \(A(50)\) and \(A(500)\).
  3. Why is there no vertical asymptote in the realistic domain?
