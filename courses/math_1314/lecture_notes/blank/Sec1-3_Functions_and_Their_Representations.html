Section 1.3

Functions and Their Representations

Student Notes · View worked solutions →

Today we introduce the concept of a function — the central object of study in this course. We’ll see four ways to represent a function (verbally, numerically, symbolically, graphically), how to describe its inputs and outputs using domain and range, and how to tell whether a graph or a relation actually represents a function.

What is a Function?

Definition — Function

A function is a rule that assigns to each input exactly one output. The set of all valid inputs is the domain, and the set of all resulting outputs is the range.

Function Notation

We write \(y = f(x)\), read “\(y\) equals \(f\) of \(x\).” Here \(x\) is the independent variable (the input), and \(y\) is the dependent variable (the output). The function \(f\) takes \(x\) and produces \(y\).

Example: if \(f(x) = 3x + 1\), then \(f(2) = 3(2) + 1 = 7\).

Example 1

Function notation

Let \(g(x) = 2x + 5\). Evaluate each of the following.

  1. \(g(3)\)
  2. \(g(-4)\)
  3. \(g(0)\)

Domain and Range

Domain and Range

The domain of a function is the set of all valid inputs. The range is the set of all outputs that the function actually produces.

Example 2

Domain and range from a set of ordered pairs

Let \(f\) be the function defined by

\(f(-1) = 4, \quad f(0) = 3, \quad f(1) = 4, \quad f(2) = -2\).

  1. Write \(f\) as a set of ordered pairs.
  2. Give the domain and range of \(f\).

Set-Builder and Interval Notation

When the domain or range is a set of real numbers (rather than a finite list), we describe it using either set-builder or interval notation.

Set-Builder Notation

\(\{x \mid \text{condition on } x\}\) reads “the set of all \(x\) such that condition.”

For example, \(\{x \mid x \geq 2\}\) is the set of all real numbers greater than or equal to 2.

Interval Notation
  • Use a bracket [ or ] when the endpoint is included.
  • Use a parenthesis ( or ) when the endpoint is not included.
  • \(\infty\) and \(-\infty\) are not numbers, so always pair them with parentheses.

Examples:

  • \([2, \infty)\) means \(x \geq 2\).
  • \((2, 5]\) means \(2 < x \leq 5\).
  • \((-\infty, 3)\) means \(x < 3\).
  • \((-\infty, \infty)\) means all real numbers.
Example 3

Translating between notations

Express each set in both interval notation and set-builder notation.

  1. All real numbers strictly less than 4.
  2. All real numbers from \(-3\) (included) up to 7 (not included).
  3. All real numbers \(x\) with \(x \leq -1\) or \(x \geq 5\).

Four Ways to Represent a Function

Every function can usually be represented in more than one way. The four standard representations are:

  • Verbal — described in words.
  • Numerical — given as a table of values.
  • Symbolic — written as a formula.
  • Graphical — drawn as a graph in the \(xy\)-plane.

Identifying Functions

Not every relation is a function. Recall that a function must give exactly one output for each input.

Theorem — Vertical Line Test

A graph in the \(xy\)-plane represents \(y\) as a function of \(x\) if and only if every vertical line crosses the graph in at most one point.

Wrap-Up

  • A function assigns exactly one output to each input.
  • The four representations — verbal, numerical, symbolic, graphical — are different views of the same rule.
  • Use set-builder or interval notation to describe domains and ranges of real numbers.
  • Use the vertical line test to decide whether a graph represents a function.
