Quiz 3

Linear Inequalities and Modeling

Worked Solutions

Covers Sections 2.3, 2.4

Instructions. This is a take-home quiz. You may use your lecture notes and online resources, but the work you submit must be your own. Show your work clearly to receive credit.

Total: 10 points

Problem 1 [2 pts]

Solve and write the answer in interval notation. (1 pt each)

  1. \(2x - 7 \leq 5\)
  2. \(3 - 4x > 19\)

(a) \(x \leq 6\). Solution: \((-\infty, 6]\).

(b) \(-4x > 16\); divide by \(-4\) (flip): \(x < -4\). Solution: \((-\infty, -4)\).

Problem 2 [2 pts]

Solve the three-part inequality and write the answer in interval notation: \(-5 \leq 3x + 1 < 7\).

Subtract 1: \(-6 \leq 3x < 6\). Divide by 3: \(-2 \leq x < 2\). Solution: \([-2, 2)\).

Problem 3 [2 pts]

The pressure \(P\) (atm) of a gas sample varies directly with depth \(d\) (ft) below the lake’s surface. At 33 ft the pressure is 2 atm.

  1. Find the constant of variation. (1 pt)
  2. What is the pressure at 99 ft? (1 pt)

(a) \(P = kd\). \(2 = 33k\), so \(k = \tfrac{2}{33}\) atm/ft.

(b) \(P = \tfrac{2}{33}(99) = 6\) atm.

Problem 4 [2 pts]

A printing service charges a fixed setup fee plus a per-page price. 50 pages cost \$40, and 200 pages cost \$115. Find a linear model \(C(p)\) for the total cost of \(p\) pages.

Slope: \(m = \dfrac{115 - 40}{200 - 50} = 0.5\). Using \((50, 40)\): \(40 = 0.5(50) + b\), so \(b = 15\). Model: \(C(p) = 0.5 p + 15\).

Problem 5 [2 pts]

Let \(\displaystyle f(x) = \begin{cases} 3 - x & \text{if } x < 0 \\ x^2 + 1 & \text{if } 0 \leq x \leq 3 \\ 2x & \text{if } x > 3 \end{cases}\). Evaluate \(f(-2)\) and \(f(4)\). (1 pt each)

\(f(-2) = 5\) (use first piece). \(f(4) = 8\) (use third piece).
