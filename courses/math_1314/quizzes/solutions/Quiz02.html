Quiz 2

Lines and Linear Equations

Worked Solutions

Covers Sections 2.1, 2.2

Instructions. This is a take-home quiz. You may use your lecture notes and online resources, but the work you submit must be your own. Show your work clearly to receive credit.

Problem 1

Write the equation of the line in slope-intercept form.

  1. Slope \(-2\), \(y\)-intercept \((0, 5)\).
  2. Slope \(\tfrac{3}{4}\), passes through \((4, -1)\).
  3. Through \((-2, 3)\) and \((6, -1)\).

(a) \(y = -2x + 5\).

(b) Point-slope: \(y + 1 = \tfrac{3}{4}(x - 4)\), so \(y = \tfrac{3}{4}x - 4\).

(c) Slope \(\dfrac{-1 - 3}{6 - (-2)} = -\tfrac{1}{2}\). Use \((-2, 3)\): \(y - 3 = -\tfrac{1}{2}(x + 2)\), so \(y = -\tfrac{1}{2}x + 2\).

Problem 2

Find the equation of the line through \((3, -2)\) that is

  1. parallel to \(y = 4x - 1\),
  2. perpendicular to \(y = 4x - 1\).

(a) Slope 4: \(y + 2 = 4(x - 3)\), so \(y = 4x - 14\).

(b) Perpendicular slope \(-\tfrac{1}{4}\): \(y + 2 = -\tfrac{1}{4}(x - 3)\), so \(y = -\tfrac{1}{4}x - \tfrac{5}{4}\).

Problem 3

Solve \(5(x - 3) - 2x = 4x + 7\).

Distribute: \(5x - 15 - 2x = 4x + 7\), so \(3x - 15 = 4x + 7\). Subtract \(3x\): \(-15 = x + 7\), so \(x = -22\).

Problem 4

Solve \(\dfrac{x}{4} - \dfrac{x - 1}{6} = \dfrac{5}{12}\).

Multiply by 12: \(3x - 2(x - 1) = 5\). Distribute: \(3x - 2x + 2 = 5\), so \(x = 3\).

Problem 5

Decide whether each is conditional, an identity, or a contradiction.

  1. \(4(x + 2) = 4x + 8\)
  2. \(2x - 5 = 2x + 1\)
  3. \(7x - 3 = 11\)

(a) Identity (true for all \(x\)).

(b) Contradiction — no solution.

(c) Conditional, \(x = 2\).

Problem 6

A car rental costs \$25 per day plus \$0.20 per mile.

  1. Write a linear function for the cost \(C\) of renting for one day and driving \(m\) miles.
  2. For what number of miles would the cost be \$65?

(a) \(C(m) = 25 + 0.20 m\).

(b) Solve \(25 + 0.20 m = 65\): \(0.20 m = 40\), so \(m = 200\) miles.
