Quiz 8

Rational Functions and Equations

Worked Solutions

Covers Sections 4.6, 4.7

Instructions. This is a take-home quiz. You may use your lecture notes and online resources, but the work you submit must be your own. Show your work clearly to receive credit.

Problem 1

Find the domain of each function.

  1. \(f(x) = \dfrac{2x}{x^2 - 16}\)
  2. \(g(x) = \dfrac{x + 1}{x^2 + 9}\)

(a) \(x^2 - 16 = 0\) at \(x = \pm 4\). Domain: \((-\infty, -4) \cup (-4, 4) \cup (4, \infty)\).

(b) Denominator is always positive. Domain: all real numbers.

Problem 2

Find any vertical asymptotes, holes, and the horizontal asymptote of each.

  1. \(f(x) = \dfrac{x - 3}{x^2 - 9}\)
  2. \(g(x) = \dfrac{2x^2 - 5}{x^2 + 1}\)

(a) \(\dfrac{x - 3}{(x - 3)(x + 3)} = \dfrac{1}{x + 3}\) for \(x \neq 3\). VA at \(x = -3\); hole at \(x = 3\); HA \(y = 0\) (numerator degree less than denominator).

(b) No real zeros of denominator, so no VA. HA: equal degrees, leading coefficients 2 and 1; HA \(y = 2\).

Problem 3

Solve \(\dfrac{3}{x - 2} - \dfrac{1}{x} = 1\). Check for extraneous solutions.

Multiply both sides by \(x(x - 2)\): \(3x - (x - 2) = x(x - 2)\), so \(3x - x + 2 = x^2 - 2x\), \(2x + 2 = x^2 - 2x\), \(x^2 - 4x - 2 = 0\). Quadratic formula: \(x = \dfrac{4 \pm \sqrt{24}}{2} = 2 \pm \sqrt{6}\). Neither is \(0\) or \(2\), so both valid.

Problem 4

Solve \(\sqrt{2x + 9} = x + 3\).

Square: \(2x + 9 = x^2 + 6x + 9\), so \(0 = x^2 + 4x\), \(x(x + 4) = 0\), \(x = 0\) or \(x = -4\).

Check: \(x = 0\): \(\sqrt{9} = 3\) ✓. \(x = -4\): \(\sqrt{1} = 1\) but \(-4 + 3 = -1\) ✗.

Solution: \(x = 0\).

Problem 5

Solve \(\dfrac{x + 4}{x - 2} \geq 0\). Write the answer in interval notation.

Boundaries: \(x = -4\) (numerator) and \(x = 2\) (denominator, excluded).

Test: \(x = -5\): \(-1 / -7 > 0\) ✓. \(x = 0\): \(4 / -2 < 0\) ✗. \(x = 3\): \(7 / 1 > 0\) ✓.

Solution: \((-\infty, -4] \cup (2, \infty)\).
