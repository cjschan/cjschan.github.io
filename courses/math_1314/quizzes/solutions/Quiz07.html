Quiz 7

Polynomial Functions and Real Zeros

Worked Solutions

Covers Sections 4.2, 4.4

Instructions. This is a take-home quiz. You may use your lecture notes and online resources, but the work you submit must be your own. Show your work clearly to receive credit.

Total: 10 points

Problem 1 [3 pts]

For \(f(x) = -2x^4 + 3x^2 - x + 1\), state (1 pt each)

  1. the degree,
  2. the leading coefficient,
  3. the end behavior in words.

(a) 4. (b) \(-2\). (c) Even degree, negative leading coefficient: both ends fall to \(-\infty\).

Problem 2 [2 pts]

Find each zero and its multiplicity for \(f(x) = (x - 5)^2(x + 2)^3(x - 1)\). For each zero, indicate whether the graph crosses or touches the \(x\)-axis.

\(x = 5\) (mult. 2, touches), \(x = -2\) (mult. 3, crosses), \(x = 1\) (mult. 1, crosses).

Problem 3 [2 pts]

Let \(f(x) = x^3 + 4x^2 + x - 6\). Use the factor theorem to decide which of the candidates \(x = 1\), \(x = -1\), \(x = -2\), \(x = -3\) are zeros of \(f\). Then write \(f(x)\) as a product of linear factors.

Test each candidate.

  • \(f(1) = 1 + 4 + 1 - 6 = 0\) ✓
  • \(f(-1) = -1 + 4 - 1 - 6 = -4\) ✗
  • \(f(-2) = -8 + 16 - 2 - 6 = 0\) ✓
  • \(f(-3) = -27 + 36 - 3 - 6 = 0\) ✓

Three zeros for a degree-3 polynomial accounts for all of them, so \(f(x) = (x - 1)(x + 2)(x + 3)\).

Problem 4 [3 pts]

Find all real zeros of \(f(x) = x^3 + 2x^2 - 9x - 18\) by factoring (try grouping).

Group: \(x^3 + 2x^2 - 9x - 18 = x^2(x + 2) - 9(x + 2) = (x + 2)(x^2 - 9)\).

Difference of squares: \(x^2 - 9 = (x - 3)(x + 3)\). So \(f(x) = (x + 2)(x - 3)(x + 3)\). Zeros: \(x = -2, 3, -3\).
